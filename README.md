**專案簡介**: 這是個非常簡單的 HTML 範例專案，示範基本的頁面結構、內嵌樣式（CSS）與一段簡單的 JavaScript 互動。

**主要特色**:

- **範例標題**: `我的第一個網頁`，展示標題、段落、清單及按鈕。
- **內嵌 CSS**: 在 `test.html` 的 `<style>` 區塊內有基本的版面與按鈕樣式。
- **簡單互動**: `showAlert()` 函式會在按鈕被點擊時顯示 alert 訊息。

**專案內容（檔案）**:

- `test.html`: 範例網頁主檔案，包含 HTML、CSS 與 JavaScript。
- `README.md`: 本檔案，說明如何使用與調整此範例。

**如何執行（在本機）**:

- 使用瀏覽器直接開啟 `test.html`：在檔案總管中雙擊或右鍵選擇以瀏覽器開啟。
- 在 PowerShell 中可執行：

```
ii .\test.html
```

或

```
Start-Process .\test.html
```

**如何使用**:

- 進入頁面後，閱讀文字與清單項目。
- 點擊頁面中的「點擊我試試」按鈕，會觸發瀏覽器的提示視窗：

```
alert("哈囉！你觸發了 JavaScript 動態效果。");
```

**要點與可調整處**:

- 修改文字內容：直接編輯 `test.html` 中的 HTML 標籤內容。
- 調整樣式：在 `test.html` 的 `<style>` 區塊修改 CSS（字體、顏色、間距等）。
- 新增互動：在 `<script>` 區塊新增或修改 JavaScript 函式，並將事件綁到按鈕或其他元素上。

**開發備註**:

- 此專案無需額外相依套件或本機伺服器，純靜態 HTML 即可運作。
- 若要在本機以簡易伺服器檢視（可避免某些瀏覽器的本地資源限制），可使用 Python（若已安裝）：

```
# Python 3
python -m http.server 8000
# 然後在瀏覽器開啟 http://localhost:8000/test.html
```
